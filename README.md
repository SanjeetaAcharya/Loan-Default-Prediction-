# Loan-Default-Prediction-
# 🏦 Loan Default Prediction System

A machine learning web application that predicts whether a loan applicant is likely to default, built with Python and Streamlit.



## 📌 Features
- Predicts loan default risk from applicant financial data
- Choose from 5 different ML models (KNN, Decision Tree, Logistic Regression, Naive Bayes, Random Forest)
- Visual probability chart
- Sample data button for quick testing
- Color-coded risk results

## 🛠️ Tech Stack
- **Frontend/UI:** Streamlit
- **ML Models:** Scikit-learn
- **Data Processing:** Pandas, NumPy
- **Language:** Python

## 📊 Models Available
| Model | Description |
|-------|-------------|
| KNN | K-Nearest Neighbors |
| Decision Tree | Tree-based classifier |
| Logistic Regression | Linear classifier |
| Naive Bayes | Probabilistic classifier |
| Random Forest | Ensemble method |

## ⚙️ How to Run Locally
```bash
git clone https://github.com/SanjeetaAcharya/Loan-Default-Prediction-
cd Loan-Default-Prediction-
pip install -r requirements.txt
streamlit run app.py
```
